# FASTR Modules
Description:

This repository contains R scripts and test data for conducting:

A/ Data Quality Assessment using the FASTR methodology - 01_module_data_quality_assessment.R

B/ Dynamic Adjustments (for outliers and completeness) using the FASTR methodology - 02_module_data_quality_adjustments.R

C/ Disruptions Analysis using the FASTR methodology - 03_module_service_utilization.R

D/ Coverage Analysis using the FASTR methodology - 04_module_coverage_estimates.R
